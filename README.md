# vibe-treasure-hunt

How It Works
🎮 Player Flow
STEP
WHAT HAPPENS
1	
Team enters name & size → hunt begins, timer starts
2	
Clue 1 appears (300 pts). 4 wrong attempts → Clue 2 unlocks (200 pts)
3	
4 more wrong → Clue 3 (100 pts). 4 more wrong → location revealed (0 pts)
4	
Correct guess → location name revealed + Maps opens for navigation
5	
GPS verifies arrival (within 50m) → next clue unlocks automatically
6	
After location 5 → final score screen with confetti 🎉
 

⭐ Scoring
Clue 1 correct → 300 pts
Clue 2 correct → 200 pts
Clue 3 correct → 100 pts
Revealed → 0 pts
Speed bonus → up to +500 pts (full bonus under 30 mins)
